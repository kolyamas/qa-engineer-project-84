Проект ручного тестирования интернет-магазина направлен на выявление дефектов в функциональности, юзабилити и безопасности.
Цель –  создать целостный и позитивный опыт совершения покупок.
Фокусирование на проверке ключевых функций: поиск, корзина, оформление заказа, оплата. 
Результ - получение подробного отчёта о найденных дефектах и рекомендации по их устранению.
